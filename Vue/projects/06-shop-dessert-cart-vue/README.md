
# Vue 3 Dessert Shop

  

Shop app with cart

  

- Add and remove goods in cart

- Check and delete goods from cart

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
